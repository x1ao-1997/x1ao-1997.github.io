遇到的问题

在创建完善自己的github的过程中，我遇到了如下问题：
1._config.yml文件配置完成后执行命令hexo deploy时提示

not found: git

解决办法：执行命令

npm install hexo-deployer-git –save

  • 出现如下提示

commend not found

错误原因 执行命令是未进入博客目录 输入如下命令即可解决

cd blog
