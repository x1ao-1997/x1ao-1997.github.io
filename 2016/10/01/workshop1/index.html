课程收获

一、创建了自己的静态博客 https://x1ao-1997.github.io
二、安装更换主题iceman

主题（iceman）安装方法
1.Install $ git clone https://github.com/wizicer/iceman.git themes/iceman
2.Install dependence
$ npm install hexo-renderer-ejs --save
$ npm install hexo-renderer-stylus --save
$ npm install hexo-renderer-marked --save
3.Enable Modify theme setting in blog folder_config.yml to iceman.
4.Update cd themes/iceman
git pull

自定义主题色调（\themes\iceman\source\css_base\variable.styl）

更改前
//Color
color-background = #C7D6D6
color-blue = #4fa6cb
color-theme = #458585
color-font-nav = #BEE5D2
color-footer = #333832

更改后

八、在文章中插入图片、音频、超链接、书签等丰富文章表现形式

一些基本的MArkdown语法：

  • 类 Atx 形式标题 # 这是 H1 ## 这是 H2 ###### 这是 H6
  • 区块引用 Blockquotes： 在每行的最前面加上 >
  • 在 Markdown 中建立代码区块： 缩进 4 个空格或是 1 个制表符
  • 分割线 —
  • 链接 行内式：This link 
    参考式：[an example][id]   [id]: http://example.com/ “example”
  • 强调：用*或_包围字词
  • 标记代码：用反引号`将字词包围
  • 图片：
  • 反斜杠可帮助插入普通的符号

课程体会

遇到问题

  • _config.yml文件配置完成后执行命令hexo deploy时提示：not found: git
    解决办法：执行命令

npm install hexo-deployer-git --save

  • 出现如下提示：commend not found
    错误原因 执行命令是未进入博客目录 输入如下命令即可解决：

cd blog

  • 页面布局错误，无法加载CSS（如下图）
    原因：网络问题
  • 文章格式与预期不一致（如下图）
    原因：空格的错误使用
  • 向文章中添加图片时，网页链接可以

    无法显示（解决办法：暂未找到）
